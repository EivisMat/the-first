\# THE FIRST !



A Discord bot built in Python using \[Pycord](https://docs.pycord.dev/).



\---



\## Getting started



\### Prerequisites



\- Python 3.10+

\- A Discord bot token (\[create one here](https://discord.com/developers/applications))



\### Setup



1\. Clone the repo

&#x20;  ```

&#x20;  git clone https://github.com/EivisMat/the-first.git

&#x20;  cd the-first

&#x20;  ```



2\. Install dependencies

&#x20;  ```

&#x20;  pip install -r requirements.txt

&#x20;  ```



3\. Copy `.env.example` to `.env` and fill in your token

&#x20;  ```

&#x20;  cp .env.example .env

&#x20;  ```



4\. Run the bot

&#x20;  ```

&#x20;  python main.py

&#x20;  ```



\---



\## Contributing



1\. Create a branch for your change (`feature/your-feature-name`)

2\. Make your changes and commit them

3\. Open a pull request to `main`

4\. Wait for a review — all changes require approval before merging



Direct pushes to `main` are not allowed.
